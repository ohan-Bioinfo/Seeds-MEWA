# MEWA Seed Center - Pages Structure

## Navigation Structure

### Main Navigation (Header)
1. **Home** - Dashboard with map and stats
2. **Data Catalog** - Browse all seed passport data
3. **Genomics Hub** - Genetic data and analysis
4. **Seeders Exchange** - Request and share seeds
5. **Research Hub** - Publications and collaboration
6. **Community** - Researchers and users
7. **Contact** - Contact forms and support

## Page Descriptions

### 1. Home (Dashboard) - EXISTING
- Interactive map of Saudi regions
- Summary statistics cards
- Quick filters by crop and region
- Data table preview

### 2. Data Catalog - EXISTING (enhance)
- Full searchable table
- Advanced filters
- Export functionality
- Individual sample detail views

### 3. Genomics Hub - NEW
**Purpose**: Scientific genetic data access and analysis tools

**Sections**:
- Genetic markers database
- DNA sequence data downloads
- Analysis tools and protocols
- Genomic diversity visualizations
- Scientific publications related to genetics

### 4. Seeders Exchange - NEW
**Purpose**: Seed request and exchange platform for researchers

**Sections**:
- Browse available seeds
- Submit seed request form
- Track request status
- Exchange guidelines and protocols
- Material Transfer Agreements (MTA)

### 5. Research Hub - NEW
**Purpose**: Research collaboration and knowledge sharing

**Sections**:
- Research projects directory
- Scientific publications
- Collaboration opportunities
- Breeding programs
- Research protocols and methodologies

### 6. Community - NEW
**Purpose**: Connect researchers and seed center users

**Sections**:
- Researchers directory
- Institutions and partners
- User profiles (placeholder)
- Discussion forums (future)
- Events and workshops

### 7. Contact - NEW
**Purpose**: Contact forms and support

**Sections**:
- General inquiry form
- Technical support
- Collaboration requests
- Department contacts
- Location and office hours

## Color Scheme (MEWA Green Palette)

Based on the MEWA logo:
- **Primary Green**: Deep forest green (from logo background) - #0B5F3A
- **Gold Accent**: Golden yellow (from logo palm tree and emblem) - #D4AF37
- **Light Green**: Sage green for backgrounds - #E8F3EC
- **Dark Text**: Charcoal green - #1A3A2E
- **White/Cream**: Off-white backgrounds - #FAF9F6
